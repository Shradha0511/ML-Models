# Employee Attrition Prediction | ML Model Comparison

An experiment focused on building and comparing multiple machine learning models for employee attrition prediction on an imbalanced HR dataset.

## Highlights
- Built an end-to-end tabular ML workflow from preprocessing to evaluation
- Compared 6 models: Linear Regression, Logistic Regression, Decision Tree, Random Forest, SVM, and XGBoost
- Focused on business-relevant metrics for imbalance: Precision, Recall, F1, ROC-AUC, and Confusion Matrix
- Performed model-specific tuning (class weighting, threshold tuning, depth and estimator analysis, kernel and hyperparameter analysis, boosting and learning-rate analysis)
- Interpreted model behavior, tradeoffs, and failure cases instead of reporting accuracy alone

## What I Demonstrated
- Strong understanding of imbalanced classification
- Practical model selection based on minority-class performance
- Clear metric-driven reasoning and interpretability-focused analysis
- Ability to communicate technical findings in a structured, decision-oriented way

## Outcome
This project demonstrates applied ML skills across baseline, linear, tree-based, kernel-based, and boosting models, with emphasis on real-world evaluation and model comparison.
